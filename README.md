# cybersecurity-homelab

This repository serves as my (Security Tester) technical portfolio and learning log in building my pentesting environment.

## Lab Specifications
| Komponen         | Detail                          |
|------------------|---------------------------------|
| Hypervisor       | VirtualBox 7.1+                 |
| Target OS        | Ubuntu Server 24.04 LTS         |
| RAM (VM)         | 4 GB                            |
| Services         | Apache2, PHP 8.2, MySQL 8.0     |
| Vulnerable App   | DVWA (latest master branch)     |
| Access           | DVWA + phpMyAdmin via browser   |


## Prerequisites
• Hardware: PC with at least 8GB of RAM (16GB+ recommended for multiple VMs), multi-core processor, and 50GB+ of disk space.

• Software:
Latest VirtualBox (download from https://www.virtualbox.org/, version 7.1.x as of 2025).

Ubuntu Server LTS ISO (version 24.04 or later, download from https://ubuntu.com/download/server, size ~3GB).
